# Task: Browser debug API — Alistigo.version and Alistigo.about

## Description
Expose debug functions on the UMD global per ADR 0009:
- Alistigo.version() → { version, locale, buildTime, dependencies }
- Alistigo.about() → version() + { runtime, monitoring, analytics }

Both print to console.log and return the object.

## Acceptance Criteria
- [ ] Alistigo.version() returns correct version from package.json
- [ ] Alistigo.version() lists bundled dependency versions
- [ ] Alistigo.about() includes runtime.storageType and runtime.mountedContainers
- [ ] Alistigo.about() includes monitoring.sentry.enabled and analytics.posthog.enabled
- [ ] Both functions callable from browser DevTools console
- [ ] Unit tests pass for debug.ts

## Technical Details
- src/debug.ts exports version() and about() — pure functions
- about() reads from a shared runtime state singleton (mountedContainers registry)
- Versions injected at build time via import.meta.env
