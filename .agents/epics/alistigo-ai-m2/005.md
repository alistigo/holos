# Task: Rename & restructure app to alistigo-artifact-playground

## Description

Rename `apps/alistigo-list-embedded-app` → `apps/alistigo-artifact-playground` and
restructure it into a two-page Vite app: a host page and an iframe page.

The list artifact rendering that currently lives in the single-page app is moved to the
iframe page. The host page is scaffolded (empty form shell) — the Dev Config Form UI is
built in task 006.

## Acceptance Criteria

- [ ] Directory renamed: `apps/alistigo-list-embedded-app` → `apps/alistigo-artifact-playground`
- [ ] `package.json` name updated: `alistigo-artifact-playground`
- [ ] `project.json` updated with new app name and paths
- [ ] Vite config updated to multi-page build:
  - `index.html` → host page (placeholder React shell)
  - `iframe.html` → iframe page (existing list artifact rendering, retargeted)
- [ ] `nx.json` / workspace references updated if any refer to `alistigo-list-embedded-app`
- [ ] Existing list artifact renders correctly on the iframe page at `/iframe.html`
- [ ] `pnpm build` passes for the renamed app

## Technical Details

Multi-page Vite config pattern:
```ts
// vite.config.ts
export default defineConfig({
  build: {
    rollupOptions: {
      input: {
        main: resolve(__dirname, 'index.html'),
        iframe: resolve(__dirname, 'iframe.html'),
      }
    }
  }
})
```

The existing `src/` structure can remain flat for now — task 006 will introduce
`src/host/` and `src/iframe/` as the form and iframe page grow.
