## Description

first-class architectural principle. This documents the pattern that every `@alistigo`
artifact must follow, making it discoverable for future artifact authors.

## Acceptance Criteria

- [ ] `projects/alistigo-ai/architecture.md` contains a new section: **Artifact Contract — Config-Doc & State-Doc**
- [ ] Section covers:
  - Definition of the **config document** (structure, delivery mechanisms: DOM element, URL param, postMessage)
  - Minimum config schema: `{ "app": "<artifact-name>", "lang"?: "<locale>" }`
  - Definition of the **state document** (structure, load/save mechanisms: localStorage, postMessage)
  - Round-trippability requirement: exporting and re-importing the state document must reproduce the same artifact state
