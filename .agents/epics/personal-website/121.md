# Task: GitHub Actions deploy + custom domain setup

## Description

Create the GitHub Actions CI/CD workflow that builds the Astro site on every push to `main` and deploys to the `gh-pages` branch. Configure the custom domain `mikael-labrut.alistigo.com` via `CNAME` file. Document the DNS setup required.

## Acceptance Criteria
- [ ] `.github/workflows/personal-website.yml` exists and runs on push to `main`
- [ ] Workflow steps: checkout → pnpm install → `nx run personal-website:build` → deploy to `gh-pages`
- [ ] Uses `peaceiris/actions-gh-pages@v3` for deployment
- [ ] `apps/personal-website/public/CNAME` contains `mikael-labrut.alistigo.com` (already checked in task 001, verify here)
- [ ] Workflow only runs when files under `apps/personal-website/` or `data/resume.json` change (path filter)
- [ ] `apps/personal-website/README.md` documents DNS setup: which A records / CNAME to add for `mikael-labrut.alistigo.com → <username>.github.io`
- [ ] GitHub repo Pages setting instructions documented (Settings → Pages → Source: `gh-pages` branch)

## Technical Details

Workflow file (`.github/workflows/personal-website.yml`):
```yaml
name: Deploy personal-website
on:
  push:
    branches: [main]
    paths:
      - 'apps/personal-website/**'
      - 'data/resume.json'
jobs:
  deploy:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: actions/checkout@v4
      - uses: pnpm/action-setup@v4
      - uses: actions/setup-node@v4
        with: { node-version: '22', cache: 'pnpm' }
      - run: pnpm install --frozen-lockfile
      - run: pnpm exec nx run personal-website:build
      - uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./apps/personal-website/dist
```

DNS for subdomain CNAME: `mikael-labrut.alistigo.com CNAME MLKiiwy.github.io`
