# Task: Hero + About + Contact tiles

## Description

Implement three bento tiles: Hero (full-width, name + role + tagline + CTA buttons), About (bio + photo), and Contact (email, LinkedIn, GitHub links). All data comes from `data/resume.json` (`basics` section).

## Acceptance Criteria
- [ ] `BentoHero.astro` renders: full name, role/label, summary tagline, CTA buttons (LinkedIn, GitHub, email)
- [ ] `BentoAbout.astro` renders: profile photo (`public/avatar.jpg`), short bio paragraph from `basics.summary`
- [ ] `BentoContact.astro` renders: email link, LinkedIn profile link, GitHub profile link
- [ ] All links open correctly (email uses `mailto:`, socials use `basics.profiles`)
- [ ] Tiles are styled with the design system from task 004
- [ ] Hero is full-width (spans all columns) on both mobile and desktop
- [ ] Responsive: readable on 375px mobile viewport

## Technical Details

Hero CTA buttons: primary ("View Experience" scrolls to experience tile), secondary ("GitHub" links to MLKiiwy profile).

Profile photo: `public/avatar.jpg` — placeholder can be used if real photo not yet added.

`basics.profiles` contains entries like `{ "network": "LinkedIn", "url": "..." }` and `{ "network": "GitHub", "url": "https://github.com/MLKiiwy" }`.

## Dependencies

Depends on tasks 001 (scaffold) and 004 (layout + design system).
