# Task: Sentry error monitoring in @alistigo/artifact-list

## Description
Integrate Sentry into the UMD bundle per ADR 0008:
- Add @sentry/browser and @sentry/react dependencies
- Create src/monitoring.ts — Sentry.init with DSN from VITE_SENTRY_DSN
- Wire pino error-level logs to Sentry.captureException
- Wrap App component in Sentry.ErrorBoundary

- [ ] Bundle size increase < 30 KB gzipped

- Use Sentry EU endpoint: encoded in the DSN URL itself
- ErrorBoundary fallback: render null (the widget disappears gracefully)
