# Task: Experience timeline tile

## Description

Implement the `BentoExperience.astro` tile showing work history as a vertical timeline. Data comes from `data/resume.json` (`work` array). Each entry shows company name, role/position, date range, and a one-line summary.

## Acceptance Criteria
- [ ] `BentoExperience.astro` renders all `work` entries from `resume.json`
- [ ] Each entry shows: company name, position title, date range (formatted as "Jan 2021 – Mar 2023"), summary
- [ ] Timeline visual connector between entries (CSS vertical line + dot)
- [ ] Entries sorted newest first
- [ ] "current" / open-ended roles show "Present" for end date
- [ ] Tile is scrollable internally if content overflows (or expands to fit)
- [ ] Responsive: readable on 375px mobile

## Technical Details

Date formatting: parse ISO date strings from resume.json (`"2021-01"` format) → `"Jan 2021"`. Use `Intl.DateTimeFormat` for locale-aware formatting.

The timeline dot + line is pure CSS: `position: relative` on the container, `::before` pseudo-element for the vertical line, a small circle `::before` on each entry.
