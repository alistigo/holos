# Task: Skills tile

## Description

Implement the `BentoSkills.astro` tile showing Mikael's tech stack as a tag cloud or grouped list. Data comes from `data/resume.json` (`skills` array). Skills are grouped by category (e.g. Languages, Cloud, Architecture).

## Acceptance Criteria
- [ ] `BentoSkills.astro` renders all `skills` entries from `resume.json`
- [ ] Skills are displayed as styled tags/pills
- [ ] Optional: grouped by `skills[].name` category (e.g. "Languages: TypeScript, Go, ...")
- [ ] Tags use the accent color from the design system
- [ ] Tile adapts to content height (no fixed height causing overflow)
- [ ] Responsive: readable on 375px mobile

## Technical Details

JSON Resume `skills` array: `[{ "name": "Languages", "level": "Master", "keywords": ["TypeScript", "Go", "Rust"] }]`

Each `keywords` item becomes a tag. Group tags under their `name` category heading.

Simple pill style: `bg-accent/10 text-accent border border-accent/20 rounded-full px-3 py-1 text-sm`
